Bang xep hang duoc sap theo thu tu uu tien sau:

1) Completed (cao hon xep tren)
2) Avg Steps (thap hon xep tren)
3) Avg Path (thap hon xep tren)
4) Avg Time (thap hon xep tren)

Giai thich nhanh:

- Completed: so testcase hoan thanh / tong testcase
- Avg Steps: so vong lap trung binh khi testcase thanh cong
- Avg Path: tong so buoc di chuyen trung binh khi testcase thanh cong
- Avg Time: thoi gian CPU trung binh moi testcase (ms)
